# UX Baseline Screenshots (PH2)

These screenshots are the visual regression baseline for the PH2 modernization pass.

## Files

- `ph2-meter.png`
- `ph2-scanner.png`
- `ph2-market.png`

## Capture rules

- Window width close to desktop default (`~1120x720`).
- Use default theme and no OS-level forced high-contrast mode.
- Capture after initial view render settles.
- Replace baseline files when intentional visual changes are shipped.
